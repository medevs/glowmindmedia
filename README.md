# GlowMind Media Website

## set up

1.Download and extract the template

2.**cd** into the template directory then run this command to install all the dependencies
    
```bash
    npm install
```
    
or
    
```bash
    yarn install
 ```


You can start the project on the local server
    
```bash
    npm run dev
 ```
